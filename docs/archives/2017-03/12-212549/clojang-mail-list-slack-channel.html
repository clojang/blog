Clojang Mail List & Slack Channel

New Stuff!

March 12, 2017

by oubiwann

Getting Project Communications Set Up

Mail List

A mail list for Clojang has been created and is availble for user subscriptions here:

  • https://groups.google.com/forum/#!forum/clojang

Once subscribed, you may send emails to the list via clojang@googlegroups.com.

Slack

Additionally, a Slack channel has been set up at clojang.slack.com. However, you need an invite in order to join the rooms. If you would like to participate in Clojang development or usage discussions, send a direct message to @Clojang (requires mutual follow) or open a Slack invitation request ticket.
